## 🇬🇧 English Description

Welcome to the official Android assistant application of Marmara Games! This repository contains the open-source code and infrastructure developed for our mobile ecosystem.

Formerly known as *SJ App Store*, this platform now hosts Marmara Games' official AI assistant, **SunAI**, and our expanding application hub. Powered by C++ and Java SDK architecture, this app aims to provide the highest performance and a seamless user interface.
